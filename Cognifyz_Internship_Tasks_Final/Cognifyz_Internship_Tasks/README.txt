Cognifyz Software Development Internship
--------------------------------------------------

This archive contains completed Java programs for the internship tasks.
Each task is organized by Level and Task number as per the instructions.

Contents:
=========
Level1_Task1_TextGame       - Number guessing game (TextGame.java)
Level1_Task2_NumberPattern  - Number pattern printing (NumberPattern.java)
Level2_Task3_CRUD_Console   - Console-based CRUD Task Manager (TaskManager.java)
Level2_Task4_TempConverter  - Temperature converter (TemperatureConverter.java)
Level3_Task5_CRUD_FileIO    - Task Manager with file storage (TaskFileManager.java)
Level3_Task6_WebScraper     - Simple web scraper using JSoup (SimpleScraper.java)

How to Run:
===========
1. Install JDK and set JAVA_HOME path.
2. Compile each Java file using:
   javac FileName.java
3. Run the program using:
   java ClassName

Note for Web Scraper:
----------------------
The web scraper uses JSoup library. Download jsoup-x.x.x.jar and include it in your classpath:
Example:
   javac -cp .;jsoup-1.16.1.jar SimpleScraper.java
   java -cp .;jsoup-1.16.1.jar SimpleScraper

This zip file contains only source code. No Excel or binary output files were required as per the given tasks.
